Lora font downloaded from https://fonts.google.com/specimen/Lora

The directory mini/ contains versions of the fonts in [woff] format with some
glyphs removed to reduce the size. The glyphs that remain are:

- Basic Latin
- Latin-1 Supplement
- Greek
- General Punctuation
- Super and Sub scripts
- Currency Symbols
- Letterlike Symbols
- Mathematical Operators
- Geometrical Shapes
- Math Misc. Symbols-A
- Latin Ligatures

-----
[woff]: https://developer.mozilla.org/en-US/docs/Web/Guide/WOFF
